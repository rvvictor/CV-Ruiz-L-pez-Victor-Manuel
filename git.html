GIT

Sistema de control de versiones que sirve para gestionar las versiones por las que va pasando el código de los proyectos.

INIT

Cree un repositorio Git vacío o reinicie uno existente

STATUS

Mostrar el estado del árbol de trabajo

BRANCH

Enumerar, crear o eliminar sucursales

CHECKOUT

Cambie de rama o restaure los archivos del árbol de trabajo

REMOTE

Administrar un conjunto de repositorios rastreados

PULL

Obtenga e integre con otro repositorio o una sucursal local

PUSH

Actualizar referencias remotas junto con objetos asociados

CLONE

Clonar un repositorio en un nuevo directorio

COMMIT

Registrar cambios en el repositorio
